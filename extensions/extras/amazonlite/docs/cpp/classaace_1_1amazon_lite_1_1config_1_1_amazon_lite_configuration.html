static

Factory method used to programmatically generate AmazonLite configuration data. The data generated by this method is equivalent to providing the following JSON values in a configuration file:

{
"aace.amazonLite" : {
"rootPath" : "Top level file system path where the model files (and other assets) are stored"
"models" : [
{
"locale" : "locale",
"path" : "relative path to the locale Model file"
}
]
}
}
