static EngineConfiguration com.amazon.aace.amazonlite.config.AmazonLiteConfiguration.createAmazonLiteConfig ( final String  rootPath,
final ModelConfig []  modelList 
)
static

Factory method used to programmatically generate AmazonLite configuration data. The data generated by this method is equivalent to providing the following JSON values in a configuration file:

{
"aace.amazonLite" : {
"rootPath" : "Path where all the model files (and other assets) are stored"
"models" : [
{
"locale" : "locale",
"path" : "relative path to the locale Model file"
}
]
}
}
Parameters
rootPathThe path where all the assets are stored.
modelListThe list of locale and their corresponding model filenames.
