PoiLookupResponse

Table Of Contents

  • Structure
  • Parameters
  • Example
Sent to the engine.

Asynchronously respond to a @c poiLookupRequest() query with results of the lookup.

JSON Structure

{
  "header" : {
    "version" : "3.1",
    "messageType" : "Publish",
    "id" : {{String}},
    "messageDescription" : {
      "topic" : "LocalSearchProvider",
      "action" : "PoiLookupResponse"
    }
  },
  "payload" : {
    "response" : {{String}}
  }
}

Parameters

header

Parameter Description Type
version "3.1" String
messageType "Publish" String
id A string that will act as a unique identifier for this message String
messageDescription Meta information about the message Object
messageDescription.topic "LocalSearchProvider" String
messageDescription.action "PoiLookupResponse" String

payload

Parameter Description Type
response A lookup response in the specified format. String
