◆ createFileConfig()

static EngineConfiguration com.amazon.aace.localVoiceControl.config.LocalVoiceControlConfiguration.createFileConfig ( final String  configFilePath)
static

Factory method used to programmatically generate an engine configuration for local voice control using a file path. The contents of the file referenced by the path should match the configuration file used to launch the local Alexa endpoint. The developer must ensure that the application has permissions to read the file.

The data generated is equivalent to providing the following JSON values in a configuration file:

{
"lvc" :
{
"configFilePath" : "<PATH TO THE CONFIGURATION FILE>"
}
}
Parameters
configFilePathPath to the configuration file where execution controller and platform services socket directories are defined.

Factory method used to programmatically generate an engine configuration for Local Voice Control. This method allows passing values for the specified parameters directly. The data generated by this method is equivalent to providing the following JSON values in a configuration file:

{
"lvc":
{
"controllerSocketDirectory" : "<SOCKET_DIRECTORY_PATH_FOR_EXECUTION_CONTROLLER>",
"controllerSocketPermissions" : "<SOCKET_PERMISSION>",
"platformSocketDirectory" : "<SOCKET_DIRECTORY_PATH_FOR_PLATFORM_SERVICES>",
"platformSocketPermissions" : "<SOCKET_PERMISSION>",
"address" : "<LOCAL_ALEXA_ENDPOINT_ADDRESS>",
"messageRouterSocketDirectory" : "<SOCKET_DIRECTORY_PATH_FOR_MESSAGE_ROUTER>"
}
}
Parameters
[in]controllerSocketDirectoryThe directory path where file socket will be created for execution controller.
[in]controllerSocketPermissionsThe permissions for the file socket. Supported values: OWNER - sets / checks the permission for owner only (600 for socket, 500 for directory) GROUP - sets / checks the permission for owner and group (660 for socket, 550 for directory) ALL - sets / checks the permission for owner, group, and other (666 for socket, 555 for directory)
[in]platformSocketDirectoryThe directory path where file socket will be created for platform services.
[in]platformSocketPermissionsThe permissions for the file socket. Supported values: OWNER - sets / checks the permission for owner only (600 for socket, 500 for directory) GROUP - sets / checks the permission for owner and group (660 for socket, 550 for directory) ALL - sets / checks the permission for owner, group, and other (666 for socket, 555 for directory)
[in]addressThe IP address of the local Alexa endpoint.
[in]messageRouterSocketDirectoryThe directory path to the socket file for hybrid message router and local Alexa endpoint connection
